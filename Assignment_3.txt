8) Count lower case and upper case letter. 

Code:-

def count_letters(a):
    count = 0
    count1 = 0 
    for letter in a:
        if letter.islower(): 
            count += 1
        elif letter.isupper(): 
            count1 += 1
    return count,count1

a=input("Enter your choice:") 
x,y=count_letters(a) 
print("Upper case letter is:",x) 
print("Lower case letter is:",y)
